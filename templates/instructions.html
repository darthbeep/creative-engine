Instructions

To use this app input the username of the account you want to use in the username field. If you want to use a partial caption for image descriptions, put the word or characters that come before your description in the start box, and the word or characters that come after in the end box.

For example, the account creative_engine_project always has the characters "/30" in the beginning of an image description, so this would go in the start box, and it always ends an image description with "About this project", so that can go in the end text box.

To generate based on whole instagram captions, leave these boxes blank.
