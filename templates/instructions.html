Instructions

This is an app that generates prompts for art generation based on past instagram captions.

To use this app input the username of the account you want to use in the username field. This will be the account that the generated caption will be based off of. If you consistently start or end your instagram posts with the same blocks of text, you can put in this text (or a portion thereof) in the start and end text boxes. For the start text box, put the end few words or characters of your block of text. For the end text box, put the first few words or characters of your block of text. These blocks of text should be not be contained elsewhere in the captions, so they should be somewhat unique.

For example, the account creative_engine_project always has the phrase "Image Description:" before the image description, so this would go in the start box, and it always ends an image description with "About this project", so that can go in the end text box.

To generate based on whole instagram captions, leave these boxes blank.

To use this website to generate artistic prompts, it is highly recommended to create a second Instagram account to create these descriptions. If there are not enough or long enough descriptions, this application will not work. The sample Instagram account writes a 20 word description of each piece, which is sufficient for training, whereas 5 word descriptions are unlikely to work.
